# VoidBorn-Website

WebSite for our game VoidBorn.

### How to start

`npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### How to add Documentation

Put your markdown documentation file in the folder public/docs :

```bash
public/
    docs/
        technique/
            file.md
            images/
                img.png
        lore/
            file.md
            images/
                img.png
        art-bible/
            file.md
            images/
                img.png
        etc..
```

### Example of markdown file

```md


### **Bold Title**
## Title

---
separation

#### `example of code: int i = 5;`

Here some text.

![This is an image](docs/path_to_your_img.png)
```
